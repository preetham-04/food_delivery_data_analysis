# food_delivery_data_analysis
File delivery data analysis using the dataset, this project demonstrates a real-world data integration and analysis pipeline for a food delivery platform.
Data from multiple sources and formats is combined into a single dataset for meaningful business insights.
# Technologies used:
- Python
- Pandas
- SQLite
- Jupyter Notebook
# Key Steps
- Loaded data from CSV, JSON, and SQL files
- Merged datasets using left joins
- Cleaned and prepared the final dataset
- Answered analytical questions
